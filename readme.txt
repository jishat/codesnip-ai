== Description ==
CodeSnip AI helps developers manage HTML/CSS/JS snippets inside WordPress and use GPT to clean, explain, or convert code to Tailwind.

== Privacy ==
No data is sent to external services unless you enter your own API key. No keys or data are stored on our servers.
